# 🔬 牛顿环实验测量工具

基于 **OpenCV.js** 和 **Vue 3** 的自动图像识别与曲率半径计算系统，专为大学物理实验设计。

## ✨ 核心功能

### 🎯 智能图像识别
- **自动检测圆心**：基于二值图光斑质心定位，精准识别牛顿环中心
- **径向剖面法**：72角度采样，智能聚类识别暗环
- **自适应预处理**：CLAHE对比度增强、高斯模糊去噪、自适应阈值分割
- **支持偏心圆环**：容差检测，适应实验中的微小偏移

### 📊 数据处理与计算
- **直径测量**：自动计算各暗环直径（像素/毫米）
- **曲率半径计算**：基于公式 R = (Dm² - Dn²) / [4(m-n)λ]
- **多组分组计算**：5组标准分组 (20-15, 19-14, 18-13, 17-12, 16-11)
- **平均曲率半径**：自动计算并显示最终结果

### 🖼️ 图像处理
- **可视化结果**：Canvas 叠加显示检测圆环、编号标签、中心十字
- **交互式预览**：鼠标悬停高亮对应圆环，表格联动
- **参数调试面板**：实时调整亮度、对比度、模糊、锐化等 8 项参数
- **多图片管理**：支持批量上传、切换、删除，自动缓存

### 💾 数据管理
- **SessionStorage 持久化**：检测数据自动缓存，刷新不丢失
- **CSV 导出**：完整测量数据表格，支持 Excel 打开
- **PNG 导出**：带标注的识别结果截图
- **拖拽上传**：支持点击和拖拽两种方式

## 📐 物理原理

### 牛顿环干涉
牛顿环是由平凸透镜与平面玻璃之间的空气薄膜产生的等厚干涉现象。当单色光垂直照射时，形成明暗相间的同心圆环。

### 曲率半径公式
利用暗环直径计算透镜曲率半径：

**R = (Dm² - Dn²) / [4(m - n)λ]**

其中：
- **R**：透镜曲率半径（米）
- **Dm, Dn**：第 m 环和第 n 环的直径（毫米）
- **m, n**：环的级数
- **λ**：入射光波长（钠光波长 589.3 nm）

```

├── image-manager.js         # 图片管理（上传、缓存、切换）
├── image-processor.js       # 图像处理（预处理、检测、排序）
├── canvas-drawer.js         # Canvas 渲染（绘制圆环、标签）
├── data-calculator.js       # 数据计算（直径、曲率半径）
├── interaction-handler.js   # 交互处理（悬停、拖拽）
├── constants.js             # 常量配置（物理参数、算法参数）
── lib/
    ├── vue.global.js        # Vue 3 库
    └── opencv.js            # OpenCV.js 库
```

### 核心算法流程
1. **图像预处理**：灰度化 → CLAHE增强 → 高斯模糊 → 自适应阈值 → 形态学闭运算
2. **圆心检测**：二值图轮廓查找 → 最大白色区域（光斑）质心计算
3. **暗环检测**：径向剖面采样 → 局部最小值查找 → 半径聚类 → 排序编号
4. **数据提取**：关键点定位 → 椭圆拟合 → 直径计算 → 曲率半径计算

##  故障排查

### 常见问题

**Q: 无法检测到暗环？**
- 检查图片质量：确保牛顿环清晰可见
- 调整对比度和 CLAHE 参数增强图像
- 检查像素标定值是否正确

**Q: 圆心位置偏移？**
- 系统自动检测光斑质心，通常准确
- 如偏移严重，可尝试调整图像亮度/对比度
- 确保中心光斑区域完整未被遮挡

**Q: 检测到的圆环数量不对？**
- 调整径向剖面对比度阈值（需修改代码）
- 尝试不同的模糊和锐化参数
- 确保图片分辨率足够（建议 > 800px）

**Q: 加载缓慢或卡顿？**
- OpenCV.js 首次加载约需 5-10 秒（9.7MB）
- 大图片处理可能需要 2-5 秒
- 建议图片宽度控制在 2000px 以内

### 调试技巧
- 查看浏览器控制台日志，了解检测过程
- 观察日志面板中的进度信息
- 检查检测到的圆环数量和半径数据
